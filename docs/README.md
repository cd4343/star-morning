# 文档目录说明

本目录包含项目的各种文档。

## 文档列表

| 文档 | 说明 |
|------|------|
| `ARCHITECTURE.md` | 系统架构设计文档 |
| `FEATURES.md` | 功能需求与业务规则文档 |
| `DEPLOYMENT.md` | 部署指南 |
| `CHANGELOG.md` | 更新日志 |
| `CODE_HEALTH_REPORT.md` | 代码健康检查报告 |
| `HOSTS_SETUP.md` | Hosts 文件配置说明 |

## 阅读顺序建议

1. **新成员入门**：`ARCHITECTURE.md` → `FEATURES.md`
2. **部署应用**：`DEPLOYMENT.md` → `HOSTS_SETUP.md`
3. **代码维护**：`CODE_HEALTH_REPORT.md`
4. **版本更新**：`CHANGELOG.md`
